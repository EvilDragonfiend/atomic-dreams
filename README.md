# Atomic Dreams package

This package for Atom is a port of the [Sublime Dreams](https://github.com/seiyria/sublime-dreams/commit/b1362a6633e5c6c248d3f133ced3cf133c95da24) package for Sublime text.

It was automatically converted with [this](https://discuss.atom.io/t/convert-sublime-grammar-to-atom-grammar/14843) method.
